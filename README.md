# Image Vision Explorer (ivExplorer)

**ivExplorer** is a native Windows desktop application developed in C++ using Microsoft Visual Studio. It serves as a specialized management and navigation tool for **Image Vision** data files and images.

Image Vision is a robust system used by financial institutions to capture and process high volumes of financial documents, including checks, deposit slips, and internal vouchers. This explorer provides a streamlined interface to interact with the complex data structures generated by that system.

## 🚀 Features

* **Native Performance:** Built with C++ for high-speed handling of large-scale financial datasets.
* **Data Exploration:** Navigate and search through Image Vision data files with ease.
* **Image Viewer:** Integrated viewing capabilities for captured document images (checks, slips, etc.).
* **File Management:** Tools to organize and manage the lifecycle of Image Vision assets.
* **Scalability:** Optimized to handle the massive operations typical of large banking environments.

## 🛠 Prerequisites

* **Operating System:** Windows 10 or higher.
* **Development Environment:** Microsoft Visual Studio (2019 or 2022 recommended).
* **Dependencies:** Ensure you have the C++ Desktop Development workload installed in Visual Studio.

## 🖥 Usage

After launching ivExplorer, you can:
1.  **Import:** Point the application to an Image Vision data directory.
2.  **Navigate:** Use the sidebar/grid view to browse through processed batches or individual documents.
3.  **Inspect:** Double-click a record to view the associated MICR data and document images.
